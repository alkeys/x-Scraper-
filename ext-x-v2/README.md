# ext-x-v2

Extension MV3 para capturar y descargar media de X/Twitter en un click.

## Funciones

- Empezar/detener captura
- Contador automatico de imagenes y videos
- Descargar todo directo (sin pasar por script Python)
- Descargar solo imagenes
- Descargar solo videos
- Exportar TXT de URLs
- Limpiar cache local

## Instalar en Firefox

1. Abre `about:debugging`
2. Entra a `This Firefox`
3. Click en `Load Temporary Add-on...`
4. Selecciona `manifest.json`

## Instalar en Chrome/Brave

1. Abre `chrome://extensions`
2. Activa `Developer mode`
3. Click en `Load unpacked`
4. Selecciona carpeta `ext-x-v2`

## Uso

1. Abre `x.com/<usuario>`
2. Pulsa `Empezar captura`
3. Haz scroll
4. Pulsa `Descargar todo` o usa filtros `solo imagenes` / `solo videos`

## Nota

Solo descarga media que X ya cargo en pagina.
